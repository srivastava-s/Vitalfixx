// turbo-all

1. Run the dev server:
```
npm run dev
```
Run this from the project root directory. Use `WaitMsBeforeAsync: 5000` and `RunPersistent: false`. The command runs in the background — check that it outputs "Ready" before confirming to the user.

2. Confirm to the user that the server is running at http://localhost:3000
